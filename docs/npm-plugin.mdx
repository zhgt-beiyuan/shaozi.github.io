## Nodejs 版本升级

### Mac 电脑升级 node

```bash
# 查看当前 node 版本
node -v

# 清除 npm 的缓存
sudo npm cache clean -f

# 安装 n 模块，用来升级更新 node
sudo npm install -g n

# 查看 node 的所有版本
npm view node versions

# 升级 node 到 最新版
sudo n latest

# 升级 node 到稳定版（截止 2022-05-29，稳定版本为 v16.15.0）
sudo n stable

# 升级 node 到指定版本
sudo n xx.xx
```
